[[every1]]
== every(Array<T>, (T) -> Boolean): Boolean

Returns `true` if every element in the list (an array) matches the condition.


The function stops iterating after the first negative evaluation of a list.

=== Parameters

[%header, cols="1,3"]
|===
| Name | Description
| `list` | The input list (an `Array`).
| `condition` | A condition (or expression) to apply to the input `list`.
|===

=== Example

This example applies a variety of expressions to the elements in the input arrays.

==== Source

[source,DataWeave, linenums]
----
%dw 2.0
import * from dw::core::Arrays
var arr0 = [] as Array<Number>
output application/json
---
{ "results" : [
     "ok" : [
        [1,1,1] every ($ == 1),
        [1] every ($ == 1)
     ],
     "err" : [
        [1,2,3] every ((log('should stop at 2 ==', $) mod 2) == 1),
        [1,1,0] every ($ == 1),
        [0,1,1,0] every (log('should stop at 0 ==', $) == 1),
        [1,2,3] every ($ == 1),
        arr0 every true,
     ]
   ]
 }
----
