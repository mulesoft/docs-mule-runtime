= Objects (dw::core::Objects)
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

This module contains helper functions to work with Objects.

The module is included with Mule runtime. To use it, you must import it to your
DataWeave code, for example, by adding the line `import dw::core::Objects` or
`import * from dw::core::Objects` to your header.
