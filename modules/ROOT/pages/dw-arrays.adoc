= Arrays (dw::core::Arrays)
ifndef::env-site,env-github[]
include::_attributes.adoc[]

This module contains helper functions for working with arrays.

To use this module in a Mule app, you must import it to your
DataWeave code, for example, by adding the line `import dw::core::Arrays` or
`import * from dw::core::Arrays` to your header.
