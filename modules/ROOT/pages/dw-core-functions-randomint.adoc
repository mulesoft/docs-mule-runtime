[[randomint1]]
== randomInt(Number): Number

Returns a pseudo-random integer number between 0 and the specified number
(exclusive).

.Transform
[source,dataweave,linenums]
----
%dw 2.0
output application/json
---
{
  price: randomInt(1000) //Returns an integer from 0 to 1000
}
----
