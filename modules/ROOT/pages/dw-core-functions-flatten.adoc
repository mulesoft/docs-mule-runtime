[[flatten1]]
== flatten(Array<Array<T> | Q>): Array<T | Q>

Flattens an array of arrays into a single, simple array.


=== Example

This example flattens an array of arrays.

==== Source

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
flatten(payload)
----

==== Input

[source,JSON,linenums]
----
[
   [3,5],
   [9,5],
   [154,0.3]
]
----

==== Output

[source,JSON,linenums]
----
[
  3,
  5,
  9,
  5,
  154,
  0.3
]
----
