[[islowercase1]]
== isLowerCase&#40;String&#41;: Boolean

Checks if the `text` contains only lowercase characters.


_Introduced in DataWeave 2.2.0. Supported by Mule 4.2 and later._

=== Parameters

[%header, cols="1,3"]
|===
| Name | Description
| `text` | The input string.
|===

=== Example

This example shows how `isNumeric` behaves with different inputs and sizes.

==== Source

[source,DataWeave, linenums]
----
%dw 2.0
import isLowerCase from dw::core::Strings
output application/json
---
{
  "a": isLowerCase(null),
  "b": isLowerCase(""),
  "c": isLowerCase("  "),
  "d": isLowerCase("abc"),
  "e": isLowerCase("aBC"),
  "f": isLowerCase("a c"),
  "g": isLowerCase("a1c"),
  "h": isLowerCase("a/c")
}
----

==== Output

[source,JSON,linenums]
----
{
  "a": false,
  "b": false,
  "c": false,
  "d": true,
  "e": false,
  "f": false,
  "g": false,
  "h": false
}
----
