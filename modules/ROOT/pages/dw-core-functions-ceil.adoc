[[ceil1]]
== ceil(Number): Number

Rounds a number upwards, returning the first full number above than the
one provided.

.Transform
[source,dataweave,linenums]
----
%dw 2.0
output application/json
---

{
  a: ceil(1.5),
  b: ceil(2.2),
  c: ceil(3)
}
----

.Output
[source,json,linenums]
----
{
  "a": 2,
  "b": 3,
  "c": 3
}
----
