[[joinby1]]
== joinBy(Array<Any>, String): String


Merges an array into a single string value, using the provided string as a separator between elements.

.Transform
[source,dataweave,linenums]
----
%dw 2.0
output application/json
---
aa: ["a","b","c"] joinBy "-"
----

.Output
[source,json,linenums]
----
{
  "aa": "a-b-c"
}
----
