[[frombase641]]
== fromBase64(String): Binary

Converts a Base64 string representation to binary.


== Parameters

[%header, cols="1,3"]
|===
| Name | Description
| `base64String` | The Base64 string representation to convert.
|===

== Example

This example takes a Base64 string representation and converts it to binary.

=== Source

[source,DataWeave, linenums]
----
%dw 2.0
import * from dw::core::Binaries
output application/json
---
{ "BinaryFromBase64" : fromBase64("TXVsZQ==") }
----

=== Output

[source,JSON,linenums]
----
{ "BinaryFromBase64": "Mule" }
----
