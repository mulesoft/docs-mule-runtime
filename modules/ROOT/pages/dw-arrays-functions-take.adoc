[[take1]]
== take&#40;Array<T&#62;, Number&#41;: Array<T&#62;

Selects first `n` elements. It returns an empty array when `n &lt;= 0` and the original array when `n > sizeOf(array)`.


_Introduced in DataWeave 2.2.0. Supported by Mule 4.2 and later._

=== Parameters

[%header, cols="1,3"]
|===
| Name | Description
| `array` | The left Array of elements.
| `n` | The number of elements to take.
|===

=== Example


==== Source

[source,DataWeave, linenums]
----
 %dw 2.0
 import * from dw::core::Arrays
 var users = ["Mariano", "Leandro", "Julian"]
 output application/json
 ---
 take(users, 2)

----

==== Output

[source,json,linenums]
----
[
  "Mariano",
  "Leandro"
]
----
