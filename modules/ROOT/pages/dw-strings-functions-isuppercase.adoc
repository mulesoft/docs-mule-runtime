[[isuppercase1]]
== isUpperCase&#40;text: String&#41;: Boolean

Checks if the `text` contains only uppercase characters.


_Introduced in DataWeave 2.2.0. Supported by Mule 4.2 and later._

=== Parameters

[%header, cols="1,3"]
|===
| Name | Description
| `text` | The input string.
|===

=== Example

This example shows how `isUpperCase` behaves with different inputs and sizes.

==== Source

[source,DataWeave, linenums]
----
%dw 2.0
import isUpperCase from dw::core::Strings
output application/json
---
{
  "a": isUpperCase(null),
  "b": isUpperCase(""),
  "c": isUpperCase("  "),
  "d": isUpperCase("ABC"),
  "e": isUpperCase("aBC"),
  "f": isUpperCase("A C"),
  "g": isUpperCase("A1C"),
  "h": isUpperCase("A/C")
}
----

==== Output

[source,JSON,linenums]
----
{
  "a": false,
  "b": false,
  "c": false,
  "d": true,
  "e": false,
  "f": false,
  "g": false,
  "h": false
}
----
