== tomorrow&#40;&#41;: Date

Returns the date for tomorrow as a `Date` type.

=== Example

This example shows the output of `tomorrow` function.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import tomorrow from dw::core::Dates
output application/json
---
tomorrow()
----

==== Output

[source,Json,linenums]
----
"2021-05-16"
----
