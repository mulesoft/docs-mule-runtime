DataWeave is the MuleSoft expression language for accessing and transforming data received through a Mule app. DataWeave is tightly integrated with Mule runtime, which runs the scripts and expressions in your Mule app.

* To learn about DataWeave 2.0 for Mule Runtime 4.1 and later, visit xref:4.1@mule-runtime::dataweave-language-guide.adoc[the language guide].
* To look up reference information about DataWeave 2.0 operators and functions, visit xref:4.1@mule-runtime::dw-functions.adoc[the reference guide].
* To find information about DataWeave 1.0 for Mule Runtime 3.9 and earlier, visit xref:3.9@mule-runtime::dataweave.adoc[the DataWeave 1.0 documentation].
