[[endswith1]]
== endsWith(String, String): Boolean

Returns `true` if a string ends with the input substring, `false` if not.


=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| `text` | The input string (a `String`).
| `suffix` | The suffix string to find at the end of the input string.
|===

=== Example

This example finds "no" (but not "to") at the end of "Mariano".

==== Source

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  "true_expression" : "Mariano" endsWith "no",
  "false_expression" : "Mariano" endsWith "to"
}
----

==== Output

[source,JSON,linenums]
----
{
  "true_expression": true,
  "false_expression": false
}
----
