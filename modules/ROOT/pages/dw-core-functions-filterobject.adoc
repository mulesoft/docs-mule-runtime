[[filterobject1]]
== filterObject({ (K)?: V }, (value: V, key: K, index: Number) -> Boolean): { (K)?: V }

Filters an object, keeping the key-value pairs that fulfill the `criteria`.


The criteria has three parameters: `value`, `key`, and `index`. Note that you can reference the value with `&#36;`, the key with `&#36;&#36;` and the index with `&#36;&#36;&#36;`.

=== Parameters

[%header, cols="1,3"]
|===
| Name   | Description
| `value` | The input object.
| `criteria` | The function that determines if a key-value pair is kept or not.
|===

=== Example

This example keeps only the key-value pairs if the value equals `"a"`.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
output application/json
---
{"letter1": "a", "letter2": "b"} filterObject ((value) -> value == "a")
----

==== Output

[source,JSON,linenums]
----
{
  "letter1": "a"
}
----

=== Example

This example only keeps the key-value pairs if the key starts with "letter".

==== Source

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{"letter1": "a", "letter2": "b", "id": 1} filterObject ((value, key) -> key startsWith "letter")
----
