[[today1]]
== today&#40;&#41;: Date

Returns the date for today as a `Date` type.

=== Example

This example shows the output of `today` function.

==== Source

[source,DataWeave,linenums]
----
%dw 2.0
import * from dw::core::Dates
output application/json
---
today()
----

==== Output

[source,Json,linenums]
----
"2021-05-15"
----
