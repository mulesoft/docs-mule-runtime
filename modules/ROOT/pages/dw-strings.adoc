= Strings (dw::core::Strings)
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

This module contains helper functions to work with Strings.

The module is included with Mule runtime. To use it, you must import it to your
DataWeave code, for example, by adding the line `import dw::core::Strings` or
`import * from dw::core::Strings` to your header.
