[[uuid1]]
== uuid&#40;&#41;: String

Returns a v4 UUID using random numbers as the source.


=== Example

This example generates a random v4 UUID.

==== Source

----
%dw 2.0
output application/json
---
uuid()
----

==== Output

[source,JSON,linenums]
----
"7cc64d24-f2ad-4d43-8893-fa24a0789a99"
----
